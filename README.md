# ChatGPT History Cleaner

一个用于清理 ChatGPT 历史对话轮次的 Edge/Chrome 浏览器扩展，通过清理页面上的历史对话来提升页面性能。

## 功能特性

- 🧹 **清理历史对话** - 清理页面上的历史对话轮次，只保留最近的 N 轮
- 📊 **查看对话数** - 快速查看当前对话的轮数
- ⚙️ **可配置保留轮数** - 自定义保留的对话轮数（1-100 轮）
- 🚀 **提升性能** - 清理大量历史对话后，页面响应速度显著提升
- 💾 **仅前端操作** - 只清理浏览器页面中的内容，不影响服务器端数据

## 安装方法

### 在 Edge 浏览器中安装

1. 打开 Edge 浏览器，访问 `edge://extensions/`
2. 开启右上角的"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择本项目的文件夹
5. 扩展安装完成！

### 在 Chrome 浏览器中安装

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目的文件夹
5. 扩展安装完成！

## 使用方法

1. 访问 [ChatGPT](https://chat.openai.com) 或 [chatgpt.com](https://chatgpt.com)
2. 打开一个包含多轮对话的聊天
3. 点击浏览器工具栏中的扩展图标
4. 设置要保留的对话轮数（默认 10 轮）
5. 点击"清理历史对话"按钮
6. 扩展会自动清理旧的对话轮次，只保留最近的 N 轮

### 使用场景

当对话轮数很多（比如 50+ 轮）时，ChatGPT 页面可能会变得很慢甚至失去响应。使用此扩展可以：
- 清理历史对话，只保留最近的对话（如最近 10 轮）
- 显著提升页面滚动和交互的流畅度
- 不影响服务器端的数据，刷新页面后历史对话仍会恢复

## 图标文件

请将以下尺寸的图标文件放置在 `icons/` 目录下：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素  
- `icon128.png` - 128x128 像素

如果没有图标文件，扩展仍然可以正常工作，只是工具栏图标会显示为默认图标。

## 注意事项

- 清理操作不可撤销，请谨慎使用
- 扩展需要访问 ChatGPT 网站的权限才能正常工作
- 如果 ChatGPT 更新了页面结构，可能需要更新选择器
